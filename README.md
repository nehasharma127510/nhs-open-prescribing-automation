# NHS Prescribing Data Automation

This project demonstrates how to automate downloading the latest NHS prescribing data
using a public API and prepare it for analytics (Parquet, Databricks).

## Features
- Automatically finds the latest monthly dataset via NHSBSA Open Data API
- No manual downloads
- Designed for analytics platforms (Parquet, Spark, Databricks)

## Why this project
I built this project to demonstrate data engineering skills by doing:
- API-driven ingestion
- Automation
- Analytics-ready file formats
